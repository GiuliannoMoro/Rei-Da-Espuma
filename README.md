# Rei-Da-Espuma

O Rei Da Espuma é um sistema desenvolvido em Java para auxiliar no gerenciamento de um lava-rápido.

O projeto permite cadastrar funcionários, clientes, veículos e serviços, além de consultar os serviços realizados por meio de uma interface gráfica simples e intuitiva.

O sistema conta com integração com um banco de dados PostgreSQL para armazenar todas as informações de forma persistente e organizada.

Além disso, os cadastros possuem relacionamento entre si para facilitar o uso da aplicação. Ao cadastrar um veículo, o sistema utiliza o ID do cliente previamente registrado para vinculá-lo ao seu proprietário. Da mesma forma, ao registrar um serviço, a placa do veículo é utilizada para identificar automaticamente o veículo correspondente, garantindo a integridade dos dados e reduzindo a necessidade de preenchimento manual.
